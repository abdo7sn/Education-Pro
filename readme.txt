== Description ==

Education Pro is a professional and versatile WordPress theme designed for educational institutions, online learning platforms, and academic organizations. It offers a modern and user-friendly interface, perfect for schools, colleges, universities, and online courses. The theme features a clean layout with vibrant colors, intuitive navigation, and easily customizable sections. Designed to cater to the needs of both educators and learners, Education Pro provides a seamless experience across devices with its responsive design. Key features include custom logos, event calendars, faculty profiles, and course listings. Its integration with popular WordPress plugins makes it an ideal choice for creating an interactive and engaging online presence.

== Installation ==

1. In your WordPress admin panel, navigate to Appearance > Themes and click the Add New button.
2. Click Upload Theme, choose the theme's .zip file from your computer, and click Install Now.
3. Once installed, click Activate to enable Education Pro on your site.
